SIRIUS Kernel Module
====================

The Kernel module of SIRIUS provides a modern "Inversion Of Control" control container which helps to build modular and maintainable Java applications.

Integration
-----------

Add the following dependency to you POM file:

.. code-block:: xml

    <dependency>
      <groupId>com.scireum</groupId>
      <artifactId>sirius-kernel</artifactId>
      <version>SNAPSHOT</version>
    </dependency>
